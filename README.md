# GameDev-Lab

一个边玩边学的游戏开发实验室。

## 项目 01：飞翔的小鸟（中文版）

使用原生 HTML、CSS、JavaScript 和 Canvas 制作，无需安装依赖。

### 在线试玩

https://gamedev-lab.vercel.app/

### 操作方法

- 手机：点击游戏区域
- 电脑：点击游戏区域，或按空格键 / ↑
- 穿过水管获得分数
- 最佳成绩保存在浏览器本地

### 文件结构

- `index.html`：游戏页面
- `style.css`：响应式界面
- `game.js`：游戏循环、碰撞、计分和音效
- `vercel.json`：Vercel 部署与安全响应头配置
